https://libgdx.badlogicgames.com/[LibGDX] is a gaming framework which works on the JVM.
Using it, you can create a game for mobile, desktop and web platform.

https://kotlinlang.org/[Kotlin] is a language compatible with the JVM (Java Virtual Machine) platform.
It's like Java but shorter.
